Algoritmo

¿Cómo determinamos si un numero es par o impar?

  1. Saber cuál es el numero a determinar si es par o impar
  2. Dividir el número entre 2
  3. Evaluar el residuo, con la operación del módulo:
    1. Si el residuo es 0, el número es par
    2. Si el residuo es 1, el número es impar
